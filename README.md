# 🤖 Nova - AI Assistant

Nova is a voice-powered AI assistant built with Python. She wakes up when you call her name, listens to your commands, answers your questions, opens apps on your computer, and shuts down when you say goodbye.

---

## ✨ Features

- 🎙️ **Wake Word Detection** — Say "Nova" and she wakes up from sleep mode and starts listening
- 💤 **Sleep Mode** — Nova stays quietly in the background listening only for her name
- 🗣️ **Voice Recognition** — Speak naturally and Nova understands you
- 💬 **Smart Replies** — Ask about the current time, date, day and Nova answers instantly
- 📱 **App Launcher** — Tell Nova to open any app on your computer
- 👋 **Voice Shutdown** — Just say "bye" and Nova shuts down gracefully

---

## 🛠️ Technologies Used

- **Python** — Core programming language
- **Speech Recognition** — For hearing and understanding voice commands
- **Text to Speech** — For Nova's voice responses
- **DateTime** — For time and date queries
- **OS / Subprocess** — For launching applications

---

## 🚀 How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/Rishi10-Spx/AI-Assistant.git
   cd AI-Assistant
   ```

2. Install dependencies
   ```bash
   pip install speechrecognition pyttsx3 pyaudio
   ```

3. Run Nova
   ```bash
   python nova.py
   ```

---

## 🎮 How to Use

| Command | What Nova Does |
|---|---|
| Say **"Nova"** | Wakes up from sleep mode |
| Ask **"What time is it?"** | Tells you the current time |
| Ask **"What day is it?"** | Tells you today's date |
| Say **"Open [app name]"** | Opens the app on your computer |
| Say **"Bye"** | Shuts Nova down |
